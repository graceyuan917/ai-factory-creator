## 五、技术与系统现状

### 8.1 现有系统
+ **ERP**: 订单管理、库存管理、BOM管理、MRP、财务
+ **WMS**: 仓库作业管理
+ **SFC**: 生产执行管理

### 8.2 系统痛点
+ **系统孤岛**: 各系统独立建设，数据不互通
+ **手工作业多**: 大量Excel表格、线下流程
+ **决策滞后**: 缺少实时数据分析能力
