## C1. 页面说明模板（逐页复制）

### C1.1 页面：模型库首页

**页面目标**
- 帮助用户快速查找和浏览3D资产模型
- 提供模型分类导航和智能搜索功能
- 展示热门模型和最近使用记录，提高查找效率

**入口与权限**
- 菜单路径：模型库 / 首页
- 可见角色：IE工程师、工厂规划师、3D模型工程师、系统管理员、项目管理员、访客用户
- 数据范围规则：所有用户可见公开模型，登录用户可见自己有权限的模型

**查询区域**

| 字段 | 类型 | 默认值 | 是否必填 | 校验规则 | 交互说明 |
|------|------|-------|---------|---------|---------|
| 关键词搜索 | 文本输入框 | 空 | 否 | 支持中英文、拼音缩写 | 输入时实时搜索，显示搜索建议，支持模糊匹配 |
| 分类筛选 | 树形选择器 | 全部 | 否 | 最多选择3个分类 | 展示资产类型树，支持多级选择 |
| 状态筛选 | 下拉单选 | 全部 | 否 | - | 选项：全部、激活、停用、草稿 |
| 排序方式 | 下拉单选 | 最近更新 | 否 | - | 选项：最近更新、最常使用、名称A-Z、名称Z-A |
| 高级筛选 | 折叠面板 | 收起 | 否 | - | 点击展开更多筛选条件：创建时间、创建人、文件大小等 |

**功能区域（按钮/批量）**

| 操作 | 适用状态 | 权限要求 | 是否二次确认 | 成功/失败提示 | 备注 |
|------|---------|---------|-------------|--------------|------|
| 上传模型 | 任何状态 | 模型上传权限 | 否 | 成功：跳转到上传页面<br>失败：权限不足提示 | 主操作按钮，固定在右上角 |
| 批量下载 | 至少选中1个模型 | 模型下载权限 | 否 | 成功：开始下载压缩包<br>失败：单个文件下载失败继续其他 | 支持多选后批量下载 |
| 批量启用/禁用 | 至少选中1个模型 | 模型管理权限 | 是 | 成功：状态更新成功<br>失败：部分失败显示详情 | 状态变更需二次确认 |
| 导出列表 | 任何状态 | 模型查看权限 | 否 | 成功：下载CSV文件<br>失败：导出失败提示 | 导出当前筛选条件下的模型列表 |

**列表区域**

| 列名 | 字段 | 展示规则 | 排序/筛选 | 汇总规则 | 备注 |
|------|------|---------|---------|---------|------|
| 模型名称 | model_name | 显示名称 + 缩略图（64x64） | 支持排序和筛选 | - | 点击跳转到模型详情页 |
| 模型编码 | model_code | 灰色小字显示 | 支持筛选 | - | 用于精确查找 |
| 分类 | type_name | 显示分类名称，带分类颜色标签 | 支持筛选 | - | 点击可筛选该分类 |
| 状态 | status | 状态标签：激活(绿色)、停用(灰色)、草稿(黄色) | 支持筛选 | - | 悬停显示状态说明 |
| 版本 | version_number | 显示当前版本号 | 支持排序 | - | 点击查看版本历史 |
| 文件大小 | geometry_data.file_size | 格式化显示（MB/KB） | 支持排序 | - | 从geometry_data中提取 |
| 创建时间 | created_at | 相对时间（如"2天前"）+ 完整时间悬停 | 支持排序 | - | 默认按此列倒序排列 |
| 操作 | - | 操作按钮组：查看、下载、编辑(有权限时) | - | - | 鼠标悬停显示 |

**详情/弹窗**
- **分组与字段**：
  - 基本信息组：模型名称、编码、分类、状态、描述
  - 3D预览组：交互式3D预览器，支持旋转、缩放、平移
  - 属性信息组：默认业务属性表格展示
  - 文件信息组：USD文件路径、文件大小、顶点数、面数
  - 版本历史组：版本列表，支持比较和回滚
- **必填与校验**：编辑时模型名称、编码必填，编码需符合命名规范
- **联动规则**：分类变更时，自动更新相关属性模板
- **关联跳转（相关单据/库存明细/日志）**：
  - 查看使用此模型的项目列表
  - 查看模型操作日志
  - 跳转到相关分类的其他模型

**空态/异常态/边界态**
- **无数据**：显示"暂无模型"插画，提示上传或联系管理员
- **无权限**：显示权限不足提示，并提供申请权限入口
- **搜索无结果**：显示"未找到相关模型"，提供搜索建议和清除筛选
- **加载超时**：显示加载动画，超时后提示重试或联系技术支持
- **批量操作部分成功**：显示操作结果摘要，列出成功和失败的项

**状态-操作映射（如适用）**

| 状态 | 可执行操作 | 限制条件 | 可编辑字段 | 备注 |
|------|-----------|---------|-----------|------|
| 草稿 | 编辑、删除、启用 | 只有创建人和管理员可操作 | 所有字段 | 未启用的模型不可被项目引用 |
| 激活 | 查看、下载、禁用、编辑(部分) | 被项目引用的模型不可删除 | 描述、属性、文件(新版本) | 主要使用状态 |
| 停用 | 查看、启用、删除 | 被项目引用的模型不可删除 | 无 | 临时不可用状态 |
| 归档 | 查看、恢复 | 只有管理员可操作 | 无 | 历史模型，只读 |

### C1.2 页面：工厂布局编辑工作台

**页面目标**
- 提供直观的3D工厂布局编辑环境
- 支持拖拽式设备放置和布局调整
- 实现布局设计、验证和优化的完整工作流

**入口与权限**
- 菜单路径：项目 / {项目名} / 编辑（工厂项目详情页的子标签页）
- 可见角色：IE工程师、工厂规划师、项目管理员（有编辑权限时）
- 数据范围规则：用户必须有项目的编辑权限，或项目已分享给用户
- 编辑能力说明：拥有项目编辑权限的角色可在Composer画布中编辑3D工厂布局，并可在右侧属性面板中编辑工厂实例对应的业务属性和变换属性


**查询区域**
> 3D编辑工作台以可视化操作为主，查询区域为资产库搜索和工厂实例搜索

| 字段 | 类型 | 默认值 | 交互说明 |
|------|------|-------|---------|
| 资产搜索 | 搜索框 | 空 | 在左侧资产库中实时过滤资产名称 |



**功能区域（按钮/批量）**

| 操作 | 适用状态 | 权限要求 | 是否二次确认 | 成功/失败提示 | 备注 |
|------|---------|---------|-------------|--------------|------|
| 保存 | 任何编辑状态 | 项目编辑权限 | 否 | 成功：保存成功提示<br>失败：保存失败原因 | 自动保存每5分钟一次，手动保存立即生效 |
| 验证（Validate） | 任何状态 | 项目编辑权限 | 否 | 弹出验证结果弹窗，列出通过/警告/错误项 | 对项目配置完整性和数据一致性进行校验 |
| 预览（Preview） | 任何状态 | 项目查看权限 | 否 | - | 以只读模式预览工厂布局 |
| 发布（Release） | 项目状态为完成时可用 | 项目发布权限 | 是 | 成功：发布成功，状态变更为已发布<br>失败：提示未通过验证 | 需先通过验证；发布后项目转为只读 |
| 撤销/重做 | 有操作历史时 | 项目编辑权限 | 否 | - | 支持多步撤销/重做 |
| 向工厂树添加子节点 | 选中节点时 | 项目编辑权限 | 否 | - | 在工厂树中为选中节点添加下级节点（工序/产线/设备） |
| 导出视图 | 任何状态 | 项目查看权限 | 否 | 成功：下载图片文件<br>失败：导出失败提示 | 导出当前Composer视图为PNG图片 |

**左侧面板上部：资产库**

**左侧面板下部：工厂树**

工厂树呈现4层层级结构，点击节点后在右侧展示对应属性面板：

| 层级 | 节点类型 | 图标颜色 | 配置状态标识 |
|------|---------|---------|------------|
| 1 | 工厂（Factory） | 蓝色 | 已配置（绿）/部分配置（黄）/未配置（灰）/配置错误（红） |
| 2 | 制程（Process） | 琥珀色 | 同上 |
| 3 | 产线（Line） | 绿色 | 同上 |
| 4 | 设备（Equipment） | 紫色 | 同上 |

> **配置状态说明**：已配置（绿）表示节点所有必需属性已完整配置；部分配置（黄）表示节点有部分必填属性未配置；未配置（灰）表示节点尚未配置任何属性；配置错误（红）表示节点配置存在错误或验证失败，需要修复。

> **示例**：例如，工厂树中可能包含：
> - SMT制程（已绑定，配置状态为绿色）
>   - SMT01# Line（已绑定，配置状态为绿色） - 右侧面板「Business Data Binding」显示为"已绑定"状态
>   - SMT02# Line（已绑定，配置状态为绿色） - 右侧面板「Business Data Binding」显示为"已绑定"状态
>   - SMT03# Line（未绑定，配置状态为灰色） - 右侧面板「Business Data Binding」显示为"未绑定"状态，除名称外其他字段为空
>     - SMT03# Line下属设备（未绑定，配置状态为灰色） - 右侧面板「Business Data Binding」显示为"未绑定"状态

> **节点交互**：节点悬停可操作（复制、重命名/删除）；节点单击触发三向联动（树选中 + 画布高亮 + 右侧面板更新）；节点双击钻入下一层视图；节点右键打开上下文菜单。详细规格见下方 **C1.2.2**。

| IoT配置 | **随台账自动导入**（非独立绑定类型） | 协议/驱动、IP、端口、站号、采样周期；数据采集点列表（支持增删改/导入导出）；**台账绑定完成后自动从台账数据填充，可在此查看或手动调整** |
| 事件配置 | **可选**，前置台账 | 事件规则列表（触发条件、事件名称、严重等级）；支持新增规则；台账未绑定时标签头显示锁定图标 |
| 状态监控 | **可选**，前置台账 | 监控参数阈值配置（最小值/最大值/单位）；支持新增监控规则；台账未绑定时标签头显示锁定图标 |
| 指标KPI | **可选**，前置台账 | 指标公式配置（OEE、MTBF、吞吐量、温度稳定性等）；支持新增指标；台账未绑定时标签头显示锁定图标 |

#### 业务数据绑定 — 右侧面板交互流程

> 工厂、产线、设备节点的属性面板均包含「业务数据绑定」相关区块，负责各绑定类型的单实例级别绑定操作。以下交互流程以「必需类型」（基础数据/台账）为主，可选类型（Events/Monitor/Metrics）的配置入口在属性面板各对应区块中，点击后跳转至 DataBindingPage 进行详细配置。与 DataBindingPage（工厂级批量管理）相互独立、互补。

**绑定状态机（每种绑定类型独立维护，分两套）**

**表单型**（基础数据/台账/IoT）：

| 状态 | 含义 | 触发条件 |
|------|------|---------|
| 未配置 | 无任何数据保存；或前置类型未完成（锁定显示） | 初始状态 |
| 部分配置 | 已保存部分数据，必填字段不完整 | 用户保存了不完整的表单；IoT 在台账完成后自动进入此状态 |
| 已绑定 | 所有必填字段完整，校验通过，同步正常 | 全量保存且校验通过 |
| 绑定异常 | 已绑定后出现异常 | 数据平台记录被删除/修改，或平台连接中断（异步检测） |



> **节点状态**（工厂树颜色）：由必需绑定类型（基础数据/台账）决定：未配置→灰、部分配置→黄、已绑定→绿、绑定异常→红；可选类型不影响节点颜色

**Step 1：绑定入口 — 属性面板「业务数据绑定」区块**

根据当前状态，区块展示内容如下：

**表单型绑定类型（基础数据/台账/IoT）：**

| 当前状态 | 展示内容 |
|---------|---------|
| 平台未连接 | 橙色提示条 + 提示词：“未同步基础数据，无法绑定”，绑定按钮 disabled |
| 未配置 | 说明文字（建议先绑定再同步）+ **「绑定业务数据」** 按钮 |
| 部分配置 | 黄色「部分配置」标识 + 已填字段预览 + 未完成必填项提示 + **「继续配置」** 按钮 |
| 已绑定 | 绿色「已绑定」标识 + 外部ID、最近同步时间 + 「禁用」「解除绑定」入口 |
| 绑定异常 | 红色「绑定异常」标识 + 错误原因说明 + 「重新同步」「解除绑定」入口 |

**规则型绑定类型（Events/Monitor/Metrics）：**



> 若实例属性面板中已存在对应字段的值，绑定区块预览时可直接展示当前值，便于与平台数据比对。

**Step 2：选择绑定记录（弹窗：「选择业务数据记录」）**

点击「绑定业务数据」后弹出记录选择弹窗：

| 交互要素 | 说明 |
|---------|------|
| 实体类型过滤 | 仅展示与当前节点类型匹配的平台记录（产线节点 → 产线记录；设备节点 → 设备记录） |
| 搜索 | 支持按名称、编码实时过滤 |
| 记录卡片 | 显示记录名称、编码、所属制程（产线）或关键属性字段（最多3条）|
| 选择操作 | 点击卡片 → 触发 Step 3 单实例校验；点击右上角关闭 → 取消，回到未绑定状态 |

**Step 3：单实例校验与冲突处理**

选中记录后立即执行校验（数据已预存本地，无需远程请求）：

*情况A：无冲突*
- 直接绑定，关闭弹窗，绑定状态变为「已绑定」

*情况B：检测到字段冲突（实例已有值 ≠ 平台值）*
- 弹出「字段冲突确认」弹窗，表格展示逐行差异：

| 列 | 说明 |
|----|------|
| 字段 | 冲突字段名称 |
| 当前值 | 实例中已存在的值（带删除线显示） |
| 平台数据（将覆盖） | 平台记录中的值（高亮显示） |

- **「确认绑定（使用平台数据）」**：以平台数据覆盖，绑定状态变为「已绑定」
- **「取消」**：关闭弹窗，绑定状态保持「未绑定」，不做任何写入

*情况C：对于设备来讲，绑定的设备数据中的所属产线并不是当前选中设备所属的产线*
- 若展示的可选设备数据，已经经过了选中设备的所属产线过滤，则不会出现此情况
- 若出现，则弹出「产线不一致，请调整当前设备的所属产线，或者绑定为目前所属产线下的设备」弹窗

> 校验范围为单一实例，仅验证当前节点的必填字段完整性和字段值一致性。工厂级整体校验（数量一致性、跨实例关系）由顶部工具栏「Validate」按钮触发，两者相互独立。

**Step 4：绑定异常（异步状态变更）**

绑定完成后，以下情况可触发状态变为「绑定异常」：
- 基础数据平台中对应记录被删除
- 关键标识字段（编码、名称）被修改导致无法匹配
- 平台连接中断

异常时，工厂树中对应节点配置状态标识变为红色，右侧面板绑定区块显示红色状态 + 错误原因，工程师需手动排查后重新绑定或解除。

**Step 5：解除绑定**

| 触发方式 | 交互说明 |
|---------|---------|
| 手动解除 | 在绑定区块点击「解除绑定」→ 二次确认提示（说明解除后数据将不再同步）→ 确认后绑定记录消失，区块恢复「未绑定」展示 |
| 自动解除（停用实例） | 停用实例前弹二次确认，明确告知「停用后将同步解除数据绑定关系」；用户确认后执行 |
| 自动解除（删除实例） | 删除实例前弹二次确认，明确告知「删除后将同步解除数据绑定关系」；用户确认后执行 |

> 实例的激活/停用状态与是否绑定数据相互独立——停用仅影响实例在场景中的可见性，不影响数据绑定状态本身；仅在停用/删除操作触发时才自动解除绑定。

**Step 6：绑定完成且无异常**
工厂树的节点状态标记为“已绑定”

### C1.3 页面：数据绑定配置页

**页面目标**
- 配置工厂项目中资产实例与业务数据的绑定关系（基于基础数据平台）

### C1.2.2 工厂树·画布·右侧面板三向联动规格

#### 概述

三个区域形成联动闭环，任意一方变动，另外两方同步更新：

```
工厂树节点  ←→  画布  ←→  右侧面板
```

#### 视图模式定义

| 视图模式 | 说明 | 画布内容 |
|---------|------|---------|
| 楼层视图（Floor View） | 默认视图，俯视整层工厂平面 | 工厂楼层平面图，可按楼层切换（2F / 1F / B1） |
| 产线视图（Line View） | 进入某条产线内部 | 该产线内设备布局，各设备可点击选中 |
| 设备视图（Equipment View） | 进入某台设备近景 | 该设备详细展示 |

#### 导航心智模型

| 操作 | 行为 |
|------|------|
| 单击节点 | 选中节点，画布在**当前视图**内高亮对应区域，右侧面板更新 |
| 双击节点 | 钻入下一层视图（产线 → 产线视图；设备 → 设备视图） |
| 右键节点 | 打开上下文菜单（含"进入产线/设备视图"等选项） |
| 点击父级节点 | 退回上层视图 |
| 点击同级节点 | 平移到对应内容（保持当前视图层级） |
| 点击面包屑父级 | 直接跳转到对应层级视图 |

#### 工厂树单击 → 画布联动规则

| 当前视图 | 点击节点类型 | 画布行为 | 右侧面板 |
|---------|------------|---------|---------|
| 楼层视图 | 工厂 | 楼层边框闪烁高亮 | 显示工厂信息 |
| 楼层视图 | 制程 | 制程区域蓝色高亮 | 显示制程信息 |
| 楼层视图 | 产线 | 产线地板区域绿色高亮 | 显示产线信息 |
| 楼层视图 | 设备 | 高亮所属产线区域 + callout pin 标注设备位置 | 显示设备信息 |
| 产线视图 | 工厂 / 制程 | 退回楼层视图，高亮对应区域 | 显示对应信息 |
| 产线视图 | 当前产线下设备 | 设备格子高亮（保持产线视图） | 显示设备信息 |
| 产线视图 | 其他产线 | 切换画布到新产线（保持产线视图层级） | 显示新产线信息 |
| 产线视图 | 其他产线的设备 | 切换到该设备所属产线，设备高亮 | 显示设备信息 |
| 设备视图 | 工厂 / 制程 | 退回楼层视图 | 显示对应信息 |
| 设备视图 | 产线 | 退回产线视图，显示该产线 | 显示产线信息 |
| 设备视图 | 任意设备 | 切换到该设备（保持设备视图层级） | 显示设备信息 |

#### 工厂树双击 → 钻入下一层视图

| 双击节点类型 | 行为 |
|------------|------|
| 产线 | 切换到产线视图，画布显示该产线内设备布局 |
| 设备 | 切换到设备视图，画布显示该设备近景 |
| 工厂 / 制程 | 不响应双击（无下一层视图） |

#### 画布点击 → 树联动规则

| 当前视图 | 点击画布区域 | 树联动（自动展开并高亮对应节点） | 右侧面板 |
|---------|------------|-------------------------------|---------|
| 楼层视图 | 楼层边框 / 空白区域 | 选中工厂节点 | 显示工厂信息 |
| 楼层视图 | 制程区域 | 选中对应制程节点 | 显示制程信息 |
| 楼层视图 | 产线地板区域 | 选中对应产线节点 | 显示产线信息 |
| 楼层视图 | 设备 callout pin | 选中对应设备节点 | 显示设备信息 |
| 产线视图 | 设备格子 | 选中对应设备节点 | 显示设备信息 |
| 产线视图 | 空白区域 | 选中当前产线节点 | 显示产线信息 |
| 设备视图 | 空白区域 / 设备本体 | 选中当前设备节点 | 显示设备信息 |

> **自动展开**：画布点击触发树联动时，若目标节点被折叠，自动展开其所有祖先节点并滚动显示该节点。

#### 面包屑导航

画布顶部始终显示当前视图层级的面包屑路径：

| 视图模式 | 面包屑示例 |
|---------|----------|
| 楼层视图 | `Houston P9 AI Factory` |
| 产线视图 | `Houston P9 AI Factory  >  SMT制程  >  SMT01# Line` |
| 设备视图 | `Houston P9 AI Factory  >  SMT制程  >  SMT01# Line  >  贴片机-A` |

- 点击面包屑中的任意父级，直接返回对应层级视图
- 面包屑各段同时触发对应节点在树中的选中和右侧面板更新

#### 多楼层自动跳层

楼层视图下，点击某产线/设备节点，若其所在楼层与当前显示楼层不同，自动切换到正确楼层并高亮目标区域，同时在楼层指示器处短暂提示"已切换到 xF"。

#### 右键上下文菜单

在工厂树节点上右键弹出上下文菜单：

| 节点类型 | 菜单项 |
|---------|-------|
| 产线 | 进入产线视图 / 重命名 / 复制 / 删除 |
| 设备 | 进入设备视图 / 重命名 / 复制 / 删除 |
| 工厂 | 重命名 / 添加子节点 |
| 制程 | 重命名 / 添加子节点 / 删除 |
