## C1. 页面说明模板（逐页复制）

### C1.1 页面：模型库首页

**页面目标**
- 帮助用户快速查找和浏览3D资产模型
- 提供模型分类导航和智能搜索功能
- 展示热门模型和最近使用记录，提高查找效率

**入口与权限**
- 菜单路径：模型库 / 首页
- 可见角色：IE工程师、工厂规划师、3D模型工程师、系统管理员、项目管理员、访客用户
- 数据范围规则：所有用户可见公开模型，登录用户可见自己有权限的模型

**查询区域**

| 字段 | 类型 | 默认值 | 是否必填 | 校验规则 | 交互说明 |
|------|------|-------|---------|---------|---------|
| 关键词搜索 | 文本输入框 | 空 | 否 | 支持中英文、拼音缩写 | 输入时实时搜索，显示搜索建议，支持模糊匹配 |
| 分类筛选 | 树形选择器 | 全部 | 否 | 最多选择3个分类 | 展示资产类型树，支持多级选择 |
| 状态筛选 | 下拉单选 | 全部 | 否 | - | 选项：全部、激活、停用、草稿 |
| 排序方式 | 下拉单选 | 最近更新 | 否 | - | 选项：最近更新、最常使用、名称A-Z、名称Z-A |
| 高级筛选 | 折叠面板 | 收起 | 否 | - | 点击展开更多筛选条件：创建时间、创建人、文件大小等 |

**功能区域（按钮/批量）**

| 操作 | 适用状态 | 权限要求 | 是否二次确认 | 成功/失败提示 | 备注 |
|------|---------|---------|-------------|--------------|------|
| 上传模型 | 任何状态 | 模型上传权限 | 否 | 成功：跳转到上传页面<br>失败：权限不足提示 | 主操作按钮，固定在右上角 |
| 批量下载 | 至少选中1个模型 | 模型下载权限 | 否 | 成功：开始下载压缩包<br>失败：单个文件下载失败继续其他 | 支持多选后批量下载 |
| 批量启用/禁用 | 至少选中1个模型 | 模型管理权限 | 是 | 成功：状态更新成功<br>失败：部分失败显示详情 | 状态变更需二次确认 |
| 导出列表 | 任何状态 | 模型查看权限 | 否 | 成功：下载CSV文件<br>失败：导出失败提示 | 导出当前筛选条件下的模型列表 |

**列表区域**

| 列名 | 字段 | 展示规则 | 排序/筛选 | 汇总规则 | 备注 |
|------|------|---------|---------|---------|------|
| 模型名称 | model_name | 显示名称 + 缩略图（64x64） | 支持排序和筛选 | - | 点击跳转到模型详情页 |
| 模型编码 | model_code | 灰色小字显示 | 支持筛选 | - | 用于精确查找 |
| 分类 | type_name | 显示分类名称，带分类颜色标签 | 支持筛选 | - | 点击可筛选该分类 |
| 状态 | status | 状态标签：激活(绿色)、停用(灰色)、草稿(黄色) | 支持筛选 | - | 悬停显示状态说明 |
| 版本 | version_number | 显示当前版本号 | 支持排序 | - | 点击查看版本历史 |
| 文件大小 | geometry_data.file_size | 格式化显示（MB/KB） | 支持排序 | - | 从geometry_data中提取 |
| 创建时间 | created_at | 相对时间（如"2天前"）+ 完整时间悬停 | 支持排序 | - | 默认按此列倒序排列 |
| 操作 | - | 操作按钮组：查看、下载、编辑(有权限时) | - | - | 鼠标悬停显示 |

**详情/弹窗**
- **分组与字段**：
  - 基本信息组：模型名称、编码、分类、状态、描述
  - 3D预览组：交互式3D预览器，支持旋转、缩放、平移
  - 属性信息组：默认业务属性表格展示
  - 文件信息组：USD文件路径、文件大小、顶点数、面数
  - 版本历史组：版本列表，支持比较和回滚
- **必填与校验**：编辑时模型名称、编码必填，编码需符合命名规范
- **联动规则**：分类变更时，自动更新相关属性模板
- **关联跳转（相关单据/库存明细/日志）**：
  - 查看使用此模型的项目列表
  - 查看模型操作日志
  - 跳转到相关分类的其他模型

**空态/异常态/边界态**
- **无数据**：显示"暂无模型"插画，提示上传或联系管理员
- **无权限**：显示权限不足提示，并提供申请权限入口
- **搜索无结果**：显示"未找到相关模型"，提供搜索建议和清除筛选
- **加载超时**：显示加载动画，超时后提示重试或联系技术支持
- **批量操作部分成功**：显示操作结果摘要，列出成功和失败的项

**状态-操作映射（如适用）**

| 状态 | 可执行操作 | 限制条件 | 可编辑字段 | 备注 |
|------|-----------|---------|-----------|------|
| 草稿 | 编辑、删除、启用 | 只有创建人和管理员可操作 | 所有字段 | 未启用的模型不可被项目引用 |
| 激活 | 查看、下载、禁用、编辑(部分) | 被项目引用的模型不可删除 | 描述、属性、文件(新版本) | 主要使用状态 |
| 停用 | 查看、启用、删除 | 被项目引用的模型不可删除 | 无 | 临时不可用状态 |
| 归档 | 查看、恢复 | 只有管理员可操作 | 无 | 历史模型，只读 |

### C1.2 页面：工厂布局编辑工作台

**页面目标**
- 提供直观的3D工厂布局编辑环境
- 支持拖拽式设备放置和布局调整
- 实现布局设计、验证和优化的完整工作流

**入口与权限**
- 菜单路径：项目 / {项目名} / 编辑（工厂项目详情页的子标签页）
- 可见角色：IE工程师、工厂规划师、项目管理员（有编辑权限时）
- 数据范围规则：用户必须有项目的编辑权限，或项目已分享给用户
- 编辑能力说明：拥有项目编辑权限的角色可在Composer画布中编辑3D工厂布局，并可在右侧属性面板中编辑工厂实例对应的业务属性和变换属性

**视口标签页**
> 中央画布区域分两个标签页，可在顶部切换

| 标签 | 说明 |
|------|------|
| Composer（3D画布） | 默认视图，展示工厂三维布局；可从资产库拖入实例、在画布中选择节点触发右侧属性面板 |
| 数据配置（Data Config） | 切换后为当前选中节点提供内联数据参数配置表单 |

**查询区域**
> 3D编辑工作台以可视化操作为主，查询区域为资产库搜索和画布工具栏

| 字段 | 类型 | 默认值 | 交互说明 |
|------|------|-------|---------|
| 资产搜索 | 搜索框 | 空 | 在左侧资产库中实时过滤资产名称 |
| 选择工具 | 工具栏按钮 | 激活 | 点击选中画布中的节点 |
| 平移工具 | 工具栏按钮 | - | 拖动平移画布视角 |
| 旋转工具 | 工具栏按钮 | - | 旋转画布视角 |
| 缩放工具 | 工具栏按钮 | - | 缩放画布 |
| 全屏 | 工具栏按钮 | - | 最大化Composer画布 |
| 网格显示 | 工具栏按钮 | 开启 | 切换场景网格显示/隐藏 |

**功能区域（按钮/批量）**

| 操作 | 适用状态 | 权限要求 | 是否二次确认 | 成功/失败提示 | 备注 |
|------|---------|---------|-------------|--------------|------|
| 保存 | 任何编辑状态 | 项目编辑权限 | 否 | 成功：保存成功提示<br>失败：保存失败原因 | 自动保存每5分钟一次，手动保存立即生效 |
| 验证（Validate） | 任何状态 | 项目编辑权限 | 否 | 弹出验证结果弹窗，列出通过/警告/错误项 | 对项目配置完整性和数据一致性进行校验 |
| 预览（Preview） | 任何状态 | 项目查看权限 | 否 | - | 以只读模式预览工厂布局 |
| 发布（Release） | 项目状态为完成时可用 | 项目发布权限 | 是 | 成功：发布成功，状态变更为已发布<br>失败：提示未通过验证 | 需先通过验证；发布后项目转为只读 |
| 撤销/重做 | 有操作历史时 | 项目编辑权限 | 否 | - | 支持多步撤销/重做 |
| 向工厂树添加子节点 | 选中节点时 | 项目编辑权限 | 否 | - | 在工厂树中为选中节点添加下级节点（工序/产线/设备） |
| 导出视图 | 任何状态 | 项目查看权限 | 否 | 成功：下载图片文件<br>失败：导出失败提示 | 导出当前Composer视图为PNG图片 |

**面板区域**
> 3D编辑工作台主要区域为Composer画布，两侧为资产库/工厂树面板和属性配置面板

**左侧面板上部：资产库**
| 区块 | 说明 |
|------|------|
| 搜索栏 | 按名称搜索资产，实时过滤 |
| 分类列表 | 设备（Equipment）、产线（Lines）、夹具（Fixtures）、厂房设施（Facility Equipment）、存储物流（Storage & Logistics） |
| 拖拽操作 | 资产卡片可直接拖入Composer画布，自动在工厂树中生成对应实例 |

**左侧面板下部：工厂树**

工厂树呈现4层层级结构，点击节点后在右侧展示对应属性面板：

| 层级 | 节点类型 | 图标颜色 | 配置状态标识 |
|------|---------|---------|------------|
| 1 | 工厂（Factory） | 蓝色 | 已配置（绿）/部分配置（黄）/未配置（灰） |
| 2 | 工序（Process） | 琥珀色 | 同上 |
| 3 | 产线（Line） | 绿色 | 同上 |
| 4 | 设备（Equipment） | 紫色 | 同上 |

节点悬停可操作：添加子节点、更多操作（重命名/删除）

**右侧面板：节点属性配置**

> 右侧面板内容随工厂树中选中的节点类型动态变化，均可编辑

**选中「工厂」节点：**
| 分组 | 主要字段 |
|------|---------|
| 基本信息 | 工厂名称、层级、ID、USD名称/ID、占地面积、地址 |
| 产能信息 | 产能、UPH（每小时产出） |

**选中「工序」节点：**
| 分组 | 主要字段 |
|------|---------|
| 基本信息 | 所属工厂层级、工序编码、工序名称、关联产线数 |
| 生产信息 | 产能、UPH（每小时产出） |

**选中「产线」节点：**
| 分组 | 主要字段 |
|------|---------|
| 基本信息 | 产线名称、编码、所属制程、备注 |
| 负责人信息 | 产线负责人、A/B/C班班长、联系方式 |
| 批次配置 | 批量大小、批次ID格式、在制品上限、追溯开关 |

**选中「设备」节点（含5个子标签页）：**

| 子标签 | 主要内容 |
|--------|---------|
| 台账（Ledger） | 基本信息（设备名称/型号/ID/位置/品牌/安装日期）、技术规格、工艺参数、备件BOM、故障知识库、操作SOP、操作记录 |
| IoT配置 | 协议/驱动、IP、端口、站号、采样周期；数据采集点列表（支持增删改/导入导出） |
| 事件配置 | 事件规则列表（触发条件、事件名称、严重等级）；支持新增规则 |
| 状态监控 | 监控参数阈值配置（最小值/最大值/单位）；支持新增监控规则 |
| 指标KPI | 指标公式配置（OEE、MTBF、吞吐量、温度稳定性等）；支持新增指标 |

**详情/弹窗**
- **分组与字段**：
  - 场景设置弹窗：网格大小、单位制、背景颜色、光照设置
  - 设备选择弹窗：分类筛选、搜索、预览、拖拽添加
  - 模板应用弹窗：选择产线模板，预览后应用到场景
  - 测量结果弹窗：显示测量距离、角度、面积等结果
- **必填与校验**：实例的业务属性必填，位置不能超出场景边界
- **联动规则**：选择数据源后，自动加载可用字段进行映射
- **关联跳转**：实例属性面板可跳转到绑定数据详情页

**空态/异常态/边界态**
- **空场景**：显示引导提示，推荐从模板开始或添加第一个设备
- **加载失败**：显示错误信息，提供重试和联系支持选项
- **权限不足**：显示只读模式，可查看但不能编辑
- **场景过大**：性能警告，建议简化场景或使用LOD模式
- **并发编辑冲突**：检测到他人修改，提供合并、覆盖或另存为选项

**状态-操作映射（如适用）**

| 状态 | 可执行操作 | 限制条件 | 可编辑字段 | 备注 |
|------|-----------|---------|-----------|------|
| 草稿（Draft） | 编辑、保存、验证、预览 | 只有项目成员可编辑 | 所有字段 | 设计阶段；验证通过后可发布 |
| 完成（Complete） | 编辑、保存、验证、预览、发布 | 只有项目成员可编辑；发布需通过验证 | 所有字段 | 可发布状态；发布后变为只读 |
| 已发布（Released） | 只读、预览、创建副本 | 不能修改原项目 | 无 | 已定稿，只读 |
| 已归档（Archived） | 只读 | 完全只读 | 无 | 历史存档 |

### C1.3 页面：数据绑定配置页

**页面目标**
- 配置工厂项目中资产实例与业务数据的绑定关系（基于基础数据平台）
- 管理三种同步场景（覆盖绑定、映射绑定、合并绑定）的配置规则
- 执行数据校验（数量一致性、关键标识符匹配、冲突检测与解决）
- 监控数据同步状态和处理异常情况

**入口与权限**
- 菜单路径：项目 / {项目名} / 数据绑定
- 可见角色：IE工程师、系统集成师、项目管理员
- 数据范围规则：用户必须有项目的编辑权限，且具有数据绑定配置权限

**查询区域**

| 字段 | 类型 | 默认值 | 是否必填 | 校验规则 | 交互说明 |
|------|------|-------|---------|---------|---------|
| 实例筛选 | 搜索选择框 | 空 | 否 | - | 按实例名称、编码搜索，支持多选 |
| 业务系统类型 | 下拉单选 | 全部 | 否 | - | 选项：全部、ERP（设备/物料）、MES（状态/工艺）、WMS（库存/库位）、IOT（传感器） |
| 绑定形式 | 下拉单选 | 全部 | 否 | - | 选项：全部、映射绑定、覆盖绑定、合并绑定 |
| 绑定状态筛选 | 下拉单选 | 全部 | 否 | - | 选项：全部、已绑定、未绑定、异常、数量不一致 |
| 同步状态筛选 | 下拉单选 | 全部 | 否 | - | 选项：全部、同步成功、同步失败、同步中、校验失败 |
| 最后同步时间 | 时间范围选择器 | 最近7天 | 否 | - | 选择时间范围筛选 |

**功能区域（按钮/批量）**

| 操作 | 适用状态 | 权限要求 | 是否二次确认 | 成功/失败提示 | 备注 |
|------|---------|---------|-------------|--------------|------|
| 添加绑定 | 选中实例时 | 数据绑定权限 | 否 | - | 打开绑定配置向导（选择绑定形式：映射/覆盖/合并） |
| 批量绑定 | 选中多个实例 | 数据绑定权限 | 是 | 成功：绑定成功数量<br>失败：失败详情（含数量差异报告） | 批量应用相同绑定规则，支持自动匹配 |

  - 连接配置：基础数据平台API端点、认证方式、请求参数、超时设置
  - 字段映射与规则：源字段-目标字段映射表，转换规则，冲突解决策略（时间戳优先/人工确认）
  - 同步场景配置：选择同步场景（场景1/2/3），配置导出/导入/双向同步策略
  - 校验配置：数量一致性校验开关，关键标识符校验规则，业务规则校验配置
  - 监控信息：最后同步时间、同步耗时、成功率、校验结果、错误历史

- **联动规则**：选择绑定形式后，自动加载对应的配置模板（映射→字段映射，覆盖→导出规则，合并→冲突解决）
- **关联跳转**：
  - 跳转到基础数据平台数据详情
  - 查看同步日志详情和校验报告
  - 跳转到相关实例的3D视图
  - 跳转到数量差异分析报告

**空态/异常态/边界态**
- **无绑定配置**：显示引导提示，说明三种同步场景的价值和配置步骤，提供场景选择向导
- **连接测试失败**：显示详细错误信息（基础数据平台连接问题），提供排查建议和重试
- **数量不一致**：显示数量差异报告（孤儿实例、缺失资产、悬空记录），提供处理建议（创建/标记/忽略）
- **数据冲突**：检测到字段值冲突，提供冲突解决界面（显示双方值、时间戳、建议解决方案）
- **校验失败**：显示校验失败详情（关键标识符不匹配、业务规则违反、格式错误），提供修复建议
- **批量操作部分成功**：显示详细结果报告（成功/失败/警告数量），支持单独重试失败项，导出差异报告
